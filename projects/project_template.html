Finance Dashboard

A dynamic Power BI dashboard created for financial reporting across three business units. This dashboard automated monthly variance reporting and introduced self-service visual analytics to non-technical teams.

View on GitHub

Project Details
This project addressed the fragmentation and time delay in financial variance reporting by centralizing data sources and enabling automated visual reporting.
Using Power BI, SQL and Python ETL scripts, the pipeline was automated end-to-end. Stakeholders could interact with live reports segmented by department, region, and financial period.
The dashboard reduced manual reporting effort by 70% and improved decision cycle time from 5 days to less than 24 hours.
