STRAVA Activity Visualisation

STRAVA offers an amazing suite of visual metrics ranging from heart rate and pace to some pretty awesome rendered GPS activity maps. While many of these insights sit behind a fairly steep paywall, STRAVA's API gives user direct access to their own data. This project taps into that opportunity, using custom-built visualisations and activity assessments to generate perspectives that even the premium subscription doesn't offer.

This project features a growing collection of custom visualisations based on my own activity data. I'll be updating it regularly as my activity log grows. Stay tuned for a fully integrated activity dashboard that will bring all my activities together!

Project Details

This project addressed the fragmentation and time delay in financial variance reporting by centralizing data sources and enabling automated visual reporting.
Using Power BI, SQL and Python ETL scripts, the pipeline was automated end-to-end. Stakeholders could interact with live reports segmented by department, region, and financial period.
The dashboard reduced manual reporting effort by 70% and improved decision cycle time from 5 days to less than 24 hours.
